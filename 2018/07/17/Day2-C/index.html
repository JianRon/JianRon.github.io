IME Starters Try-outs 2018 - C

Posted on 2018-07-17

Author: JianRon

  题意：给一个n$~(1\leqslant n \leqslant18)~$阶带边权完全图，找一条边权和最小的哈密顿路径,输出方案。
  思路：经典的状压dp，记$~dp(mask,i)~$已经经过了$~mask~$为中二进制位值为$~1~$对应的点并停在第$~i~$号结点时的最小花费。转移方程如下：$~dp(mask,i) = min{dp(mask - 2^i,j) + cost(j,i)}~,其中2^i~\&~mask = 2^i ~$
